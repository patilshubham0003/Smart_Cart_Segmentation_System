# SmartCartSegmentationSystem

## Overview
This project uses unsupervised machine learning to group customers based on their shopping behavior. It helps in understanding different types of customers.

## What I Did
- Handled missing values  
- Removed outliers  
- Scaled data using StandardScaler  
- Encoded data using OneHotEncoder  
- Used PCA for visualization  

## Algorithms Used
- K-Means Clustering  
- Agglomerative Clustering  

## Finding Best K Value
- Used Elbow Method  
- Used Silhouette Score  

## Visualization
- Used Matplotlib and Seaborn  
- Plotted clusters using PCA  

## Result
- Created different customer segments  
- Helped understand customer behavior  

## Tools and Libraries
- Python  
- Pandas  
- Scikit-learn  
- Matplotlib  
- Seaborn
